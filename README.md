## 🛠️ Desenvolvimento e Tecnologias

Este projeto é o resultado de uma evolução tecnológica que uniu lógica de programação clássica, desenvolvimento web e inteligência artificial.

### 1. 🐍 Lógica em Python
O projeto foi inicialmente concebido como um script de automação em **Python**. 
* Focado na criação de funções modulares para cálculos matemáticos.
* Utilização de estruturas condicionais (`if/elif/else`) para controle de fluxo.
* Base para a validação da lógica aritmética do sistema.

### 2. 🌐 Transposição para Web (HTML, CSS e JS)
Para tornar a ferramenta acessível e visual, a lógica original foi transposta para o ecossistema Frontend:
* **HTML5:** Estruturação semântica da calculadora.
* **CSS3:** Design responsivo e estilização visual (UI).
* **JavaScript (ES6+):** Implementação da lógica de eventos e cálculos em tempo real no navegador, substituindo a execução de terminal do Python.

### 3. 🤖 Auxílio de Inteligência Artificial
O desenvolvimento contou com o suporte da **IA (Gemini)** para:
* **Refatoração de Código:** Otimização das funções e correção de sintaxe.
* **Migração de Stack:** Auxílio na conversão da lógica Python para JavaScript de forma eficiente.
* **Documentação:** Estruturação técnica deste README e boas práticas de versionamento.
* **Tratamento de Erros:** Implementação de travas de segurança (como o bloqueio de divisão por zero e validação de caracteres não numéricos).
